# WHU-STU-CHAT
C#大作业团队仓库——武汉大学学生互助交流平台

## 版本信息
- **当前版本**: V-0.0.1T (测试版)
- **更新时间**: 2025-4-21
- **更新人**: Eaxon

## 更新内容
1. 实现用户注册与登录功能
2. 实现公共聊天室功能
   - 实时消息发送与接收
   - 在线用户列表显示
   - 系统消息通知(用户加入/离开)
3. 优化用户界面设计
4. 修复安全验证相关问题

## 技术栈
- **前端**: Vue.js + SignalR客户端
- **后端**: ASP.NET Core + SignalR
- **数据库**: MySQL

## 使用说明
1. 用户需先注册账号并登录
2. 登录后可进入公共聊天室与其他用户交流
3. 在聊天室可发送消息、查看在线用户
